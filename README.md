# Part 1 - Data Audit, EDA & Business Understanding

This repository covers raw data audit and churn-oriented exploratory analysis.

## Expected files in this repo

- `eda_audit.ipynb`
- `data_quality_report.md`
- `business_memo.md`
- `requirements.txt`

## Setup

```bash
python -m venv .venv
.venv\\Scripts\\activate
pip install -r requirements.txt
```

## Data

Create a local folder named `dataset/` and place all assignment CSV files there.

## Run

- Open and run `eda_audit.ipynb` end-to-end.
- Charts are saved in `outputs/charts/`.
- If needed, export refreshed reports from notebook cells.

## Notes

The notebook includes:
- dataset loading across all source files,
- data quality checks,
- EDA slices linked to churn,
- at least 5 churn-risk hypotheses with evidence hooks.
